## Images

### Resize
```
/800x600/d8c01b45a6.png
```

width x height

### Rotate
```
/upside|left|right/d8c01b45a6.png
```

* `upside`: 180°
* `left`: 90°
* `right`: -90°

### WebP conversion
```
/webp/d8c01b45a6.jpeg
```

### Gif to mp4
```
/mp4/d8c01b45a6.gif
```

> **Note:** Filters, resize, and rotation also work on static (non-animated) GIFs.
